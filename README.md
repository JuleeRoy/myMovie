## Introduction

This project is a simple movie streaming website that allows users to log in and explore a collection of movies. It utilizes HTML, CSS, Bootstrap, and JavaScript for the front-end. The deployment link is [My Movie Website](https://juleeroy.github.io/myMovie/).

## Features

- User authentication and login
- Responsive design using Bootstrap
- Movie showcase and information

## Deployment

The website is deployed using GitHub Pages. You can visit the deployed site [here](https://juleeroy.github.io/myMovie/).
